title: typst-wasm
description: Compile Typst documents with WebAssembly from JavaScript.
---

`typst-wasm` provides a JavaScript API for compiling Typst documents with WebAssembly.

It supports browser, Node.js, Deno, Bun, and Cloudflare Worker runtimes. You control where the WASM, workers, and fonts are loaded; the package does not download runtime assets from a CDN.

## Install

```sh
npm install typst-wasm
```

For the fonts distributed with the Typst compiler:

```sh
npm install @typst-wasm/fonts
```
