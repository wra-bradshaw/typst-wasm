Cloudflare Workers runs on `workerd`, not Node.js. Import the compiler from `typst-wasm` and use the `jspi` backend. The worker backend is unavailable in this runtime, so do not create a Web Worker or configure browser isolation headers.

<Steps>

1.  **Install the packages**

          <Tabs syncKey="package-manager">

    <TabItem label="npm">

    ```sh
    npm install typst-wasm @typst-wasm/fonts
    ```

          </TabItem>

    <TabItem label="pnpm">

    ```sh
    pnpm add typst-wasm @typst-wasm/fonts
    ```

          </TabItem>

    </Tabs>

2.  **Import the compiler and core modules**

    ```ts
    import { createTypstCompiler } from "typst-wasm";
    import core from "typst-wasm/engine/engine.core.wasm";
    import core2 from "typst-wasm/engine/engine.core2.wasm";
    import core3 from "typst-wasm/engine/engine.core3.wasm";
    import regularFontUrl from "@typst-wasm/fonts/LibertinusSerif-Regular.otf?url";
    import mathFontUrl from "@typst-wasm/fonts/NewCMMath-Regular.otf?url";
    ```

3.  **Compile inside the request handler**

    ```ts
    export default {
      async fetch(request, env) {
        const compiler = await createTypstCompiler({
          backend: "jspi",
          coreModules: {
            "engine.core.wasm": core,
            "engine.core2.wasm": core2,
            "engine.core3.wasm": core3,
          },
        });
        try {
          await compiler.addFonts(
            env.ASSETS.fetch(new URL(regularFontUrl, request.url)).then((res) =>
              res.arrayBuffer().then((res) => new Uint8Array(res)),
            ),
            env.ASSETS.fetch(new URL(mathFontUrl, request.url)).then((res) =>
              res.arrayBuffer().then((res) => new Uint8Array(res)),
            ),
          );

          await compiler.addSource("main.typ", "= Hello from Typst");
          const result = await compiler.compile({
            main: "main.typ",
            format: "html",
          });

          return new Response(result.output, {
            headers: { "content-type": "text/html; charset=utf-8" },
          });
        } finally {
          await compiler.dispose();
        }
      },
    };
    ```

</Steps>

## Fonts in Workers

`compiler.addFonts()` accepts a `Uint8Array` or a promise that resolves to one. It does not accept a URL, a `Response`, or a bare `ArrayBuffer`. The example above loads packaged fonts through the `ASSETS` binding and converts each response to `Uint8Array`.

Configure the binding in `wrangler.jsonc`:

```jsonc
{
  // ...
  "assets": { "binding": "ASSETS" }
}
```

For deployment, continue with the [Cloudflare Workers deployment guide](../../deployment/cloudflare-workers/).
