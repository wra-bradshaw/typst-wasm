---
title: Deno
description: Compile Typst documents in Deno with JSPI or Web Workers.
---

import { Aside, Steps, Tabs, TabItem } from "@astrojs/starlight/components";

Deno can run `typst-wasm` with either the JSPI backend or a browser-compatible
Web Worker. JSPI keeps compilation on the current thread; the worker backend
runs it in a dedicated worker. Choose JSPI when it is available in your Deno
version, or use the worker backend for broader compatibility.

<Steps>

1.  **Install the packages**

    ```sh
    deno add npm:typst-wasm npm:@typst-wasm/fonts
    ```

2.  **Resolve the core modules**

    ```ts
    import { createTypstCompiler } from "npm:typst-wasm";

    const asset = (specifier: string) =>
      new URL(import.meta.resolve(specifier));

    const coreModules = {
      "engine.core.wasm": WebAssembly.compileStreaming(
        fetch(asset("typst-wasm/engine/engine.core.wasm")),
      ),
      "engine.core2.wasm": WebAssembly.compileStreaming(
        fetch(asset("typst-wasm/engine/engine.core2.wasm")),
      ),
      "engine.core3.wasm": WebAssembly.compileStreaming(
        fetch(asset("typst-wasm/engine/engine.core3.wasm")),
      ),
    };
    ```

3.  **Choose a backend**

    <Tabs syncKey="deno-backend">

    <TabItem label="JSPI">

    ```ts
    const compiler = await createTypstCompiler({
      backend: "jspi",
      coreModules,
    });
    ```

    JSPI support depends on the Deno/V8 version. If this backend is not
    available, use the Web Worker example instead.

    </TabItem>

    <TabItem label="Web Worker">

    ```ts
    import { createWebWorker } from "npm:typst-wasm/worker/browser";

    const compiler = await createTypstCompiler({
      backend: "worker",
      coreModules,
      worker: () =>
        createWebWorker(asset("typst-wasm/worker/web-worker")),
    });
    ```

    </TabItem>

    </Tabs>

4.  **Load a font and compile**

    ```ts
    try {
      const font = await fetch(
        asset("@typst-wasm/fonts/NewCMMath-Regular.otf"),
      );
      await compiler.addFonts(new Uint8Array(await font.arrayBuffer()));
      await compiler.addSource("main.typ", "= Hello from Typst");

      const result = await compiler.compile({
        main: "main.typ",
        format: "pdf",
      });
      await Deno.writeFile("output.pdf", result.output);
    } finally {
      await compiler.dispose();
    }
    ```

5.  **Run the program**

    Save the example as `src/main.ts` and run it with:

    ```sh
    deno run --allow-read --allow-write --allow-net src/main.ts
    ```

</Steps>
