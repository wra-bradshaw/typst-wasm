description: Documentation for typst-wasm and its Vite plugin.

A promise based API for compiling [Typst](https://typst.app/) using wasm in as many runtimes as possible.

[Get started with typst-wasm](./packages/typst-wasm/getting-started), JavaScript bindings for compiling Typst in any runtime.

[Get started with vite-plugin-typst](./packages/vite-plugin-typst/), an experimental way to use typst-wasm as a HTML generator.

See the [typst-wasm examples](./packages/typst-wasm/examples/) and [vite-plugin-typst examples](./packages/vite-plugin-typst/examples/) for complete applications and live deployments.
