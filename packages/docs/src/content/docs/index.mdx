---
title: typst-wasm
description: Documentation for typst-wasm and its Vite plugin.
---

A promise based API for compiling [Typst](https://typst.app/) using wasm in as many runtimes as possible.

There are two main packages:

- [typst-wasm](./packages/typst-wasm/getting-started)—the main library for compiling typst.
- [vite-plugin-typst](./packages/vite-plugin-typst/)—an experimental way to use typst-wasm as a HTML generator.
