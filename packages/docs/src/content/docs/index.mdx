Compile [Typst](https://typst.app/) documents with WebAssembly from JavaScript.

- [Get started with typst-wasm](./packages/typst-wasm/)
- [Use the Vite plugin](./packages/vite-plugin-typst/)

`typst-wasm` provides the compiler API for browser, Node.js, and Cloudflare Worker runtimes. The Vite plugin builds on it and lets you import `.typ` files directly in a Vite application.
